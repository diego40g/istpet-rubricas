A1. Sistema de Biblioteca — Clase Libro y Autor (Python · Java · C#)

📝 Instrucciones para el estudiante
  • Objetivo: Implementar un sistema básico de biblioteca aplicando los conceptos fundamentales de POO: clases, objetos, atributos y métodos. El sistema gestiona dos entidades: Autor y Libro.
  • Lenguaje: Elige Python, Java o C#. La rúbrica evalúa lógica y conceptos, no la sintaxis del lenguaje.
  • Producto esperado:
    • Python: Archivo biblioteca.py con clases Autor y Libro y función main().
    • Java (IntelliJ IDEA — Maven): Archivos Autor.java, Libro.java y Main.java. Atributos private con getters. Extensión GUI: VentanaBiblioteca.java (Swing).
    • C# (JetBrains Rider — Windows Forms): Archivos Models/Autor.cs, Models/Libro.cs y Program.cs. Propiedades con { get; private set; }. Extensión GUI: Forms/FormBiblioteca.cs.
    • Estructura de proyecto recomendada: model/ → ui/ → Main (Java) / Models/ → Forms/ → Program.cs (C#).
Entrega: Sube el archivo .py, o un enlace al repositorio con los archivos .java / .cs, antes de la fecha límite. Si implementas la extensión GUI, incluye capturas de pantalla de la ventana funcionando.
