A2. Sistema de Playlist Musical — Constructores y Métodos Especiales (Python · Java · C#)

📝 Instrucciones para el estudiante
  • Objetivo: Implementar un sistema de gestión de playlists musicales aplicando constructores múltiples / alternativos, métodos especiales, colecciones y métodos estáticos.
  • Lenguaje: Elige Python, Java o C#. La rúbrica evalúa lógica y conceptos, no la sintaxis.
  • Producto esperado:
    • Python: Archivo playlist_musical.py con clases Cancion y Playlist (dunder methods + @classmethod).
    • Java (IntelliJ IDEA — Maven): Archivos Cancion.java, Playlist.java y Main.java. Sobrecarga de constructores, métodos static factory y ArrayList. Extensión GUI: VentanaPlaylist.java (Swing).
    • C# (JetBrains Rider — Windows Forms): Archivos Models/Cancion.cs, Models/Playlist.cs y Program.cs. Extensión GUI: Forms/FormPlaylist.cs.
    • Clase / script principal que demuestre: mínimo 5 canciones con distintos constructores, 2 playlists, combinar(), mezclar() y cálculo de duración total.
Entrega: Sube el archivo .py, o un enlace al repositorio con los archivos .java / .cs, antes de la fecha límite. Si implementas la extensión GUI, incluye capturas de pantalla de la ventana funcionando.
